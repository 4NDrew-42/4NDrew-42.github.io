Meet App Project Portfolio

Project Overview

Meet App is a progressive web application built using React and serverless architecture. Utilizing the test-driven development (TDD) approach, it features real-time updates and offline capabilities through the Google Calendar API.

Link to GitHub-pages deployment:

Technical Requirements

  • Serverless architecture using AWS Lambda
  • Google Calendar API for fetching event data
  • PWA features like offline support and push notifications
  • Responsive design, compatible across all modern browsers
  • High test coverage with TDD methodology

Technologies Used

This project leverages a diverse stack of modern technologies to ensure scalability, performance, and user experience. The key technologies include:

  • HTML5 & CSS3: For structuring and styling the web application.
  • React: A JavaScript library for building user interfaces dynamically.
  • Node.js: Server-side JavaScript runtime environment, used for serverless functions.
  • AWS Lambda: Provides serverless compute - executing code without provisioning servers.
  • Google Cloud: Used for additional cloud services and hosting.
  • Google Calendar API: For fetching upcoming events and integrating real-time updates.
  • OAuth2 Authentication: For secure authorization through Google's standard.
  • Service Workers: Enabling offline support and app-like features for the PWA.
  • Responsive Web Design: Ensuring the app is functional on devices of various sizes.
  • Data Visualization: Using libraries to render graphs and charts for event data.
  • Test-Driven Development (TDD): Ensuring high-quality and maintainable code.
  • Continuous Integration/Continuous Deployment (CI/CD): For automated testing and deployment.

Features & Scenarios

Key features include event filtering by city, interactive event details, and customizable event counts. Users can also install the app directly to their home screens, enhancing accessibility and user engagement.

Deployment & Monitoring

The app is hosted on GitHub Pages, with continuous integration and deployment pipelines in place. Performance monitoring is handled through an APM tool to ensure optimal user experience.

Data Visualization

Data visualizations are incorporated to present event data effectively, using scatterplots and pie charts to provide insights at a glance.
